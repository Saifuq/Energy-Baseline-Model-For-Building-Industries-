## 🔬 Project Overview
The **India AI Energy Baseline Model** is a state-of-the-art predictive analytics platform designed to decarbonize the Indian built environment. It establishes high-fidelity energy baselines using **XGBoost Gradient Boosting**, satisfying the rigorous transparency requirements of **ASHRAE Guideline 14**.

By calibrating global building patterns (ASHRAE Kaggle) with localized **NASA POWER API** and **IMD Synthetic Weather data**, the model provides 16 Indian cities with industrial-grade energy audits, BEE Star Rating projections, and Net Zero 2070 sustainability roadmaps.

### 🎯 Core Objectives
1. **Predictive Baselines**: Accurate hourly/annual kWh predictions for diverse Indian facilities.
2. **Industrial Compliance**: Automated checks against **BEE Star Ratings** and **ECBC 2017** standards.
3. **Actionable ROI**: AI-driven "What-If" simulation for HVAC, envelope, and operational upgrades.
4. **Sustainability Visualization**: Real-time CO₂ footprinting and "Net Zero" contribution metrics.

## 🛠 Technology Stack
- **Core Engine**: Python 3.9+, XGBoost (Hist-Gradient Boosting).
- **Data Engineering**: Pandas, NumPy, Scikit-Learn.
- **Frontend**: Streamlit (Premier Industrial UI/UX).
- **Visualization**: Plotly Express, Matplotlib (High-Contrast Georgia Serif theme).
- **Data Sources**: NASA POWER (Solar/Thermal), IMD (District Weather), ASHRAE Global Building Dataset.

## 🚀 Installation & Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/saifuddin-farooqui/india-ai-energy-baseline.git
   cd india-ai-energy-baseline
   ```

2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Initialize Assets**:
   Ensure you have `xgboost_india_model.json` and `indian_cities_weather_cache.csv` in the root directory.

4. **Launch Dashboard**:
   ```bash
   streamlit run app.py
   ```
